# Where's Teemo? 

#### A Where's Waldo inspired game based on League of Legends

Choose a level and find all the champions. Players are timed and their score is saved on the leaderboard. The game gives players updates for every champion selection. The leaderboard includes a profanity filter for usernames.

[View Live](https://where-is-teemo.vercel.app/)
